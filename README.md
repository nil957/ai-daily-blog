# AI Daily Blog

🤖 每日 AI 前沿资讯与深度解读

## 关于

这是一个追踪人工智能最新进展的博客，每天更新 AI 领域的热点新闻、技术突破和行业动态。

## 本地运行

```bash
gem install bundler jekyll
bundle install
bundle exec jekyll serve
```

访问 http://localhost:4000 查看博客。

## 内容涵盖

- 大语言模型 (LLM)
- AI 应用与产品
- AI 安全与治理
- AI 硬件与基础设施
